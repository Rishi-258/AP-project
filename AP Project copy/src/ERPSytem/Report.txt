Role Enforcement
User roles are stored in the database
After login, each role is redirected to its respective homepage
Each homepage only contains the buttons/features allowed for that role
Students, faculty, and admin can never see or open each other’s functions
User identity is maintained via UserSession.userId

Maintenance Mode Enforcement

A maintenance flag is stored in the database
Only the admin can toggle this via the Maintenance panel
When maintenance mode is active:
Students and faculty can open the system but cannot perform edits
All edit/submit/save buttons become disabled or hidden
Only admin retains full access


Users_auth

| Column        | Type                              | Constraints               | Description                                  |
| ------------- | --------------------------------- | ------------------------- | -------------------------------------------- |
| id            | INT                               | PK, AUTO_INCREMENT        | Unique user ID                               |
| username      | VARCHAR(50)                       | UNIQUE, NOT NULL          | Login username                               |
| password_hash | VARCHAR(255)                      | NOT NULL                  | Password (stored as plain text in your case) |
| role          | ENUM('Admin','Faculty','Student') | NOT NULL                  | User role                                    |
| created_at    | TIMESTAMP                         | DEFAULT CURRENT_TIMESTAMP | Account creation time                        |


Students

| Column     | Type         | Constraints         | Description                     |
| ---------- | ------------ | ------------------- | ------------------------------- |
| student_id | INT          | PK, AUTO_INCREMENT  | Unique student record           |
| roll_no    | VARCHAR(20)  | UNIQUE, NOT NULL    | University roll number          |
| name       | VARCHAR(100) | NOT NULL            | Student name                    |
| program    | VARCHAR(50)  | NOT NULL            | Academic program (B.Tech, etc.) |
| year       | INT          | NOT NULL            | Current academic year           |
| email      | VARCHAR(100) | Optional            | Contact                         |
| phone      | VARCHAR(15)  | Optional            | Contact                         |
| user_id    | INT          | FK → users_auth(id) | Links login to student profile  |

Faculty

| Column     | Type         | Constraints         |
| ---------- | ------------ | ------------------- |
| faculty_id | INT          | PK, AUTO_INCREMENT  |
| emp_id     | VARCHAR(20)  | UNIQUE, NOT NULL    |
| name       | VARCHAR(100) | NOT NULL            |
| department | VARCHAR(100) | NULL                |
| email      | VARCHAR(100) | NULL                |
| phone      | VARCHAR(15)  | NULL                |
| user_id    | INT          | FK → users_auth(id) |

Courses

| Column      | Type         | Constraints                                  |
| ----------- | ------------ | -------------------------------------------- |
| course_id   | INT          | PK, AUTO_INCREMENT                           |
| course_code | VARCHAR(20)  | UNIQUE, NOT NULL                             |
| course_name | VARCHAR(100) | NOT NULL                                     |
| credits     | INT          | NOT NULL                                     |
| faculty_id  | INT          | FK → faculty(faculty_id), ON DELETE SET NULL |
| capacity    | INT          | DEFAULT 60                                   |
| enrolled    | INT          | DEFAULT 0                                    |


Table Enrollment

| Column        | Type       | Constraints                                  |
| ------------- | ---------- | -------------------------------------------- |
| enrollment_id | INT        | PK, AUTO_INCREMENT                           |
| student_id    | INT        | FK → students(student_id), ON DELETE CASCADE |
| course_id     | INT        | FK → courses(course_id), ON DELETE CASCADE   |
| grade         | VARCHAR(5) | NULL                                         |
